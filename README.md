# HousingPriceKaggle

This project is predicting sales price of houses. 

1. only xgboost.XGBRegresoor (clfResultCJ.csv)
2. xgboost.XGBRegresoor after RandomizedSearchCV (regResultCJ.csv)
3. ANN(deep learning) (annResultCJ.csv)

These three models are used for training, and the second model got the best result, MSE(Mean Squared Error).
In this project, deep learning ANN got the worst result. (Deep learning seems like doesn't always get the good result)
